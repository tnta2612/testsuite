Section 1: Basic HTML Structure

HTML, or HyperText Markup Language, is the standard markup language for creating web pages. It uses tags to structure content on the web. Below is a simple example of HTML structure:


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is a simple HTML document.</p>
    </body>
</html>

Section 2: HTML Elements

HTML includes various elements for structuring content. Here are some common ones:

  • <h1> to <h6>: Header tags used for headings, with <h1> being the highest level and <h6> the lowest.
  • <p>: Represents a paragraph of text.
  • <a>: Defines a hyperlink that links to another page or resource.
  • <img>: Embeds an image into the document.
  • <ul> and <ol>: Create unordered (bulleted) and ordered (numbered) lists, respectively.
  • <table>: Defines a table, with <tr> for rows and <td> for cells.

Section 5: Advanced Topics

Advanced HTML topics include:

  • Responsive Web Design: Ensuring that web pages look good on all devices using CSS media queries.
  • HTML5 Features: New elements such as <article>, <section>, and <nav> that enhance the semantic structure of web pages.
  • Accessibility: Techniques to make web content accessible to all users, including those with disabilities.

HTML5 Example

HTML5 introduces new elements that improve the structure and semantics of web documents. Here is an example:


<article>
    <header>
        <h1>Article Title</h1>
        <p>Published on: <time datetime="2024-08-06">August 6, 2024</time></p>
    </header>
    <section>
        <h2>Introduction</h2>
        <p>This is the introduction section of the article.</p>
    </section>
    <footer>
        <p>Author: John Doe</p>
    </footer>
</article>
